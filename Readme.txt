・麻雀における清一色（染手）を練習するためのソフトです。


【起動方法】
・『chinitsu.zip』をダウンロードして任意の場所に解凍して下さい。
　（『chinitsu_tk.exe』と『data』フォルダを同じ場所に配置すること）

・『chinitsu_tk.exe』をダブルクリックすると
　「WindowsによってPCが保護されました」と警告が出ます。
　　　詳細情報 ⇒ 実行 と押下するとアプリケーションが起動します。

【説明】
・難易度のボタンを選択するとゲーム開始です。
  アガリ牌だと思う数字をすべての選択肢し、
  パネルを緑にした後に『決定』を押して下さい。

・以下のボタンは必要に応じて押して下さい。
    『理牌』：牌を種類・数字順に整列させる。（難易度ダウン）
    『洗牌』：牌をランダムに整列させる。（難易度アップ）

・音楽ファイル(mid)を再生可能です。
  ファイルを『data』フォルダに格納し、
  コンボボックスから選択後に再生ボタンを押して下さい。
